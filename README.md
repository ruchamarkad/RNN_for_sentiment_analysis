# RNN-Based Sentiment Analysis using IMDB Dataset

## 1. Goal

The goal of this project is to build a **Sentiment Analysis model** that can classify movie reviews as **positive or negative**.
We use the **IMDB movie review dataset** to train a Recurrent Neural Network (RNN) model that learns patterns in text and predicts sentiment based on the review content.
This project demonstrates how deep learning techniques can be applied to **Natural Language Processing (NLP)** tasks such as opinion mining and text classification.

---

## 2. Tools / Technologies Used

* Python 
* Pytorch
* Pandas
* scikit-Learn
* Natural Language Processing (NLP) techniques
* Recurrent Neural Network (RNN)
* IMDB Movie Review Dataset
* Jupyter Notebook / Google Colab

---

## 3. Result

The trained RNN model successfully learned patterns from the movie reviews and was able to classify sentiments with good accuracy.
The model achieved approximately **88–90% accuracy** on the test dataset, demonstrating that RNN-based architectures are effective for sentiment classification tasks.
